# BWP Documentation

This folder contains comprehensive, task-oriented documentation for the BWP (Beast Within Productions) project. If you are new here, start with the Overview and Architecture and then navigate to the topic you need.

Contents
- Overview and Architecture (what it is, how it works)
- Configuration (how to point the app to data and set tokens)
- Data Model (what is stored in configs and data/)
- API (HTTP endpoints, formats)
- Build and Deploy (Gradle, WAR, Tomcat)
- Operations (health, logs, backup/restore)
- Security (accounts, tokens, permissions, sensitive files)
- Development (project setup, tests, code style)
